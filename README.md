# 💬 Mami-chan AI — Intro Project

My first full-stack AI chatbot project, combining personality-driven design with real-world API handling.

Mami-chan is a playful, flirty AI assistant built to simulate engaging conversation while maintaining reliability even when external APIs fail.

## 🧠 What It Does
- A personality-driven AI chatbot with a custom conversational style
- Remembers basic user info during chat (like names)
- Handles API failures with built-in fallback responses to keep the experience smooth

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **API:** OpenAI API
- **Tools:** Git, GitHub

---

## ✨ Features
- 💬 Interactive chat interface
- 🎭 Custom AI personality (Mami-chan)
- 🧠 Basic memory handling (user name tracking)
- 🔁 Reset conversation feature
- ⚡ OpenAI API integration
- 🛑 Fallback system for API errors (429 / quota issues)
- 📱 Clean and responsive UI
