# Mami-chan AI 💬

A lightweight AI chatbot with memory, local fallback responses, and OpenAI integration.

Built in 48 hours 🚀

---

## ✨ Features

- 🧠 Remembers user name during conversation  
- 💬 Handles basic conversation locally (no API required)  
- 🔄 Fallback system when API fails or quota runs out  
- ⚡ Fast, short, natural responses  
- 🧹 Clear/reset conversation support  
- 🎯 Simple and clean chat UI  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Vanilla JavaScript  
- **Backend:** Node.js + Express  
- **AI:** OpenAI API (with local fallback system)
